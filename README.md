# Homebrew Tap

This repository contains Homebrew formulae for [AppCheck](https://github.com/vykes-mac/appcheck).

## Installation

```bash
brew tap vykes-mac/tap
brew install appcheck
```

Or install directly:

```bash
brew install vykes-mac/tap/appcheck
```

## Updating

```bash
brew update && brew upgrade appcheck
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `appcheck` | Local tooling to capture app walkthroughs and upload artifacts |
